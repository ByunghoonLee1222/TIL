### HTML5

* 2차원 그래픽을 위한 (canvas) 요소
* 미디어 재생 기능
* 오프라인 웹 애플리케이션 - 네트워크가 연결되지 않은 상태에서도 실행 가능
* 드래그 앤 드롭 - 요소를 마우스로 끌어서 넣을 수 있음
* 웹 스토리지 - 쿠키를 대체할 수 있는 웹 저장소 기능 제공
* 위치 정보 제공 - 지도 기능
* 웹 SQL 데이터베이스
* 파일 API 지원 - 파일 업로드와 파일 관리 기능 제공
* 웹 소켓(WebSocket) API 제공 - 서버와 브라우저 간의 양방향 통신 기능 제공
