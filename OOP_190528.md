## static import

* ```java
  import static java.lang.Math.random;
  import static java.lang.System.out;
  ```

* ```java
  System.out.println(Math.random());
  out.println(random());//다음과 같이 변경 가능
  ```

> ***배열은 가변적이지 못하고 한가지 타입만 사용 가능***(so,현업에선 잘 안쓴다)
